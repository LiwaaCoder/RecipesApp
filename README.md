# **Recipe App**

**Requirments**

    1) NodeJS version 18.16.0 or higher
    2) To run it on android emulator need to install Android Studio
    3) To run it on Iphone simulator need to install xcode

**Usage**

    in the root dir run:
    1) npm install
    2) npx expo start
    3) press `i` to run on IOS or `a` to run on Android
